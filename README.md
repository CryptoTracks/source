# CryptoTunes

## Frontend Setup - built w/ [@vue/cli](https://cli.vuejs.org/config/)
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
